# tomatenstau.de

> Personal portfolio and coding showcase website.

## 🚀 Tech Stack

- **Framework**: [Astro](https://astro.build)
- **Styling**: SCSS, FontAwesome
- **Internationalization**: Built-in Astro i18n
- **Deployment**: Static Site (GitHub Pages)

## 🛠 Usage

```bash
npm install
npm run dev
npm run build
npm run check
```
